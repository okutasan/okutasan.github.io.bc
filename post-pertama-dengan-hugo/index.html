Post Pertama Dengan Hugo

Halo, ini adalah halaman saya yang pertama menggunakan Hugo. Apa itu Hugo ?

Menurut web aslinya

“Hugo is one of the most popular open-source static site generators. With its amazing speed and flexibility, Hugo makes building websites fun again.”

bahasa gampangnya Hugo itu SSG yang cepat dan fleksibel yang mudah untuk digunakan.

hmm pasti bingung apa itu SSG(static site generators) ?

sama saya awalnya juga bingung, biasanya membuat web pakai cms semisal wordpress,joomla, ataupun drupal. Ini dapat istilah baru SSG.

secara mudahnya ssg ini adalah web page static yang tidak memerlukan page admin ataupun database karena memang tidak dinamis seperti cms. Pembuatan kontennya pun bisa secara offline dan kemudian diunggah per file. Hal ini sangat menguntungkan untuk pembuat konten yang mementingkan kecepatan dan isi dari konten. Mungkin agar lebih jelas kalian bisa cari-cari artikel di google tentang SSG.

Mungkin itu hal yang pertama dapat saya tulis di halaman pertama ini. Semoga saja saya dapat mengurus ini disela-sela waktu yang memang tidak sibuk. hehehe ….

terimakasih.
